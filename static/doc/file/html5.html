服务器对接

注册完之后进入申请房间，填写正确的信息 然后获取房间号

绑定域名可以填多个，每个域名用换行分割 。注意填写的必须和你调用的页面相符，否则连接的时候会一直保持在正在连接状态

申请房间完成后，复制连接地址填入代码就完成了

Socket.io连接服务器 
var socket = io("ws://服务器域名/你的房间号");
websocket连接服务器 
var ws = new WebSocket("ws://服务器域名/你的房间号");

客户端发送

服务器发送和接收消息主要使用socket.io,而websocket就算使用不同的API也是同时兼容的

就是说使用任意一种方式发送的消息两种方式都能同时接收

Socket.io 发送消息

和原来的一样,参数一是事件名,参数二是消息内容 

 socket.emit("事件@接收者","消息");
Websocket 发送消息

因为函数参数只能发生字符串和上面的不同 ，所以要将参数数组转字符串发送 

 ws.send('["事件@接收者","消息"]');

客户端接收

Socket.io 接收消息

直接使用官方事件绑定函数 

socket.on("事件",function(message){
    //message : "消息"
});
Websocket 接收消息

使用原生api回调函数 注意参数内容是socket.io返回的是"未序列化"的JOSN字符串 

ws.onmessage=function(str){
    //str : '["事件","消息"]' 
};
